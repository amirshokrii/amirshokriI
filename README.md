## 👋 About Me

I'm a Front-End Developer with a genuine passion for building clean, responsive, and user-friendly interfaces. I care about writing readable, maintainable code and paying close attention to the small details that make an interface feel polished — solid spacing, smooth interactions, and layouts that hold up on any screen. I'm currently deepening my skills in React and Next.js by building real-world projects, and I make it a habit to learn something new about the front-end ecosystem every day.

I'm open to new front-end opportunities where I can keep learning while contributing to real products.
